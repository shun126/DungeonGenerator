## DungeonGenerateParameter Migration Notice (Important)  
  
We have migrated `DungeonGenerateParameter` to **new enumeration values** according to the following policy:  
  
- Added **Flat** to `EDungeonExpansionPolicy`  
- Added a value equivalent to **MovePlayerStart** to `EDungeonStartLocationPolicy`  

As a result, **the old flags (`Flat` / `MovePlayerStartToStartingPoint`) are now deprecated**.  
Existing assets will be **automatically migrated to the new enumeration values** via `PostLoad`, ensuring compatibility is maintained.  

> ※The old flags are scheduled for removal in the future. Please switch to the new enumeration values whenever possible.
