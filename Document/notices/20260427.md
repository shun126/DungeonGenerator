## Upcoming DungeonGenerator 1.9.1 Release and Future Version 2.0 Plans

DungeonGenerator version 1.9.1 will be released soon.

We are also continuing development of DungeonGenerator version 2.0. Please note that version 2.0 is planned to include breaking changes.

The 1.9 series will continue to receive bug fixes as needed, so you can keep using the current version with confidence.

In version 2.0, we plan to move beyond fully random room placement and introduce level design concepts that allow each room to have a clearer purpose or role.

We also plan to review and reorganize parameters that have become difficult to maintain.

The release date for version 2.0 has not been decided yet. We will provide another notice before the release.
