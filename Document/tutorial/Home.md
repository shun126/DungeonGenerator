# Dungeon Generator

Dungeon Generator is an Unreal Engine plug-in for building room-and-aisle dungeons from configurable assets.  
It supports editor preview, runtime generation, visual theme databases, hand-authored special rooms, room sensors, and interior placement.

## English
If you want to start in English, use the links below.

- [Quick Start](./QuickStart.en.md)  
  The fastest path to generating and previewing one dungeon.
- [Tutorial Index](./README.en.md)  
  The full English guide index, including visuals, sublevels, sensors, and advanced topics.

This plug-in is intended for designers, artists, technical designers, and programmers.  
If you are new to the plug-in, start with **Quick Start** first.
